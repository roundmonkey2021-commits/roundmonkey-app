Enhance the P&L Distribution chart to support comparative analysis across phases, weeks, and months using multiple distribution curves.

Goal:

* Enable comparison of distributions across different segments
* Maintain clarity and avoid visual clutter
* Provide meaningful insights into trading behavior

---

1. COMPARISON CONTROLS

Add controls above the chart:

Step 1: "Compare By" selector:

* Options:

  * Overall
  * Phase
  * Weekly
  * Monthly

Step 2: Multi-select options:

* For Phase:

  * Phase 1, Phase 2, Phase 3...

* For Weekly:

  * Week 1, Week 2...

* For Monthly:

  * Jan, Feb...

Allow multiple selections

---

2. DEFAULT STATE

* Show single curve:

  * Overall distribution

---

3. MULTI-CURVE OVERLAY

* When multiple selections are made:

  * Plot one curve per selection

Example:

* Phase 1 → curve 1

* Phase 2 → curve 2

* Allow optional inclusion of "Overall" as baseline

---

4. COLOR SYSTEM

* Assign distinct color to each curve:

  * Phase 1 → blue
  * Phase 2 → purple
  * Phase 3 → orange

* Keep colors consistent across interactions

* Use subtle area fill (low opacity)

---

5. ZERO LINE

* Maintain a single vertical line at 0
* Shared across all curves
* Slightly stronger than grid lines

---

6. LEGEND

* Add legend showing:

  * Curve name (Phase 1, Phase 2, etc.)

* Place:

  * Top-right or inside chart

---

7. LIMIT CURVES

* Allow maximum 3–4 curves at a time
* Prevent clutter

---

8. AXIS CONSISTENCY

* Apply same axis rules:

  * Reduced label density
  * No Y-axis title
  * Clean X-axis formatting

---

9. SMOOTHING

* Apply consistent smoothing across all curves
* Ensure fair comparison

---

FINAL RESULT:

* User can compare distributions across phases/timeframes
* Clear visual differences between trading behavior
* Clean and uncluttered visualization
* Powerful insight into edge and consistency
