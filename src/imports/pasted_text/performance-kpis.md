Redesign the KPI section of the Performance Metrics page to improve hierarchy, grouping, and clarity.

Goal:

* Make KPIs visually structured
* Highlight most important metrics
* Improve scan-ability within 2 seconds

---

1. GROUP KPIs INTO SECTIONS

Create three groups:

Group 1: Performance (Primary)

* Net P&L
* Win Rate
* Profit Factor
* Expectancy

Group 2: Risk (Secondary)

* Current Drawdown
* Max Drawdown
* Lowest Point

Group 3: Context (Tertiary)

* Total Trades
* Peak P&L

---

2. LAYOUT

* Use multi-row layout instead of single row

Row 1:

* Net P&L (larger card)
* Win Rate
* Profit Factor
* Expectancy

Row 2:

* Current Drawdown
* Max Drawdown
* Lowest Point

Row 3:

* Total Trades
* Peak P&L

Spacing:

* 16px between cards
* 24–32px between rows

---

3. PRIMARY KPI EMPHASIS

* Make Net P&L visually dominant:

  * Font size: 32px
  * Bold
  * Use green/red color
  * Slightly larger card width

---

4. KPI CARD DESIGN

* Background: white
* Border: 1px solid #E5E7EB
* Border radius: 12px
* Padding: 16px

Inside each card:

* Label (top, small, grey)
* Value (bottom, bold)

---

5. TYPOGRAPHY

* Primary KPI: 30–32px bold
* Secondary KPIs: 20–24px
* Labels: 11–12px muted grey

---

6. COLOR RULES

* Positive values → green

* Negative values → red

* Neutral values → dark grey

* Do not use unnecessary colors

---

7. ALIGNMENT

* Ensure all KPI cards align in grid
* Equal spacing and consistent sizing
* Values aligned properly for readability

---

FINAL RESULT:

* KPIs feel structured and intentional
* User understands performance instantly
* Clear separation between performance, risk, and context
